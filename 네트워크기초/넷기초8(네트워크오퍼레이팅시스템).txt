 # Network Operating System
    : 소프트웨어이고, 하드웨어와 네트워크가 연결될 수 있게, 네트워크 서비스를 이용하기
      위해서 탑재가 되어야 한다. 비즈니스를 목적으로 진화해왔다.
	
 * NOS -Novell NetWare
    79년에 회사를 세움. 83년에 주식회사 상장. IPX라는 자체 프로토콜을 사용.
    인터넷 프로토콜의 호환성을 중요시하지 않음.IPX기술개발에만 집중함.마이크로소프트는
    인터넷의 중요성을 먼저 캐치함.

 * Dos - w / Apple Mac - GUI

 * NOS - Linux
   유닉스코드를 기반으로 만듦. 유닉스가 너무 볼륨이 크기에 사이즈를 줄인 버젼이다.
   오픈소스이다. 윈도와 노벨에서 제공하는 기능을 포함하고 있음. 웹서버에서 상당히 많이
   사용했음. -> 윈도우보다 훨씬 안정적이고 무료이기 때문이다.

 * NOS - UNIX
    60년대부터 개발하기 시작했음. 71년에 Bell Lab에서 처음으로 공개함. 접근성이 너무
    낮았다. 회사단위로 다루기에 어렵다... -> 유닉스를 두개로 나눔.(C.U / BSD : 상업/배포)
